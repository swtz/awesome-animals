// Algumas observações:
// No módulo numbers-animated.js é usada uma variável
// contadora que recebe um valor inicial de zero
// e ela é incrementada à medida que a callbackFn
// da função 'setInterval' é chamada.

// Essa variável precisa estar dentro do escopo
// do método .forEach, pois, A CADA CHAMADA que esse loop
// faz ao método 'setInterval', que consequentemente chama
// a função de callback, a variável contadora precisa ser
// ZERADA, pois se ela estiver no escopo do módulo, todas
// as funções farão uso dos valores adicionados a ela em tempo
// real.

// Com a variável dentro do escopo, cada chamada ao método
// 'setInterval' usará o valor de ZERO na função de callback;
// logo, ambos os números terão o mesmo comportamento, pois
// cada um usará o valor de zero como base para seu incremento.
